## Установка и запуск

* `make setup` — установка (с миграциями)
* `make serve` — запуск приложения
* `make shell` — django shell

## Дополнительно

* `make lint` — проверка линтером
* `make test` — проверка тестами
* `make test-coverage` — проверка тестами с анализом покрытия кода
